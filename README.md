# Shamba
Shamba - SmartFarming in Kenya
shamba is our solution for this hackathon. It is a tool built to help small-scale farmers in Kenya manage their land. Most weather apps just give raw data, but shamba uses AI to turn that data into a 7-day plan that is actually easy to follow.

What it does
Large UI: I styled the app with massive text and a pastel green theme. I did this so farmers can easily read the advice on their phones while outside in bright sunlight.

Soil Data: The app pulls data from the UCANR Soil-Web API. When a user enters their location, the app finds the actual soil texture (Sandy, Clay, Loam, etc.) for that specific area.

Actionable Advice: I tuned the AI prompt to focus on low-cost, realistic methods. It suggests things like using cow manure, wood ash, and mulching instead of expensive machinery or chemicals.

Swahili Support: I included a translation feature. It uses the AI to translate the plan into simple Swahili to make the tool more accessible.
